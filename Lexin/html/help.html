Help

Quick Help

Translation could be achieved in three ways:
  1. Alt + double click a word
  2. Select a word and Alt + left click it
  3. Select a word and press Lexin dictionary extension button on a toolbar

History

All words translations are stored in the history. The history page link can be found on the pop-up window and on the options page. The history is useful when you learning the language and want to store the new words in order to practice with them later. You can use the history to create a flashcards, here is how it can be used with Quizlet :

  1. Open History page
  2. Copy table with translations history (without the header) to the clipboard
  3. Open your flashcards set in Quizlet for edit
  4. Press Import data
  5. Paste translations
  6. Select “Tab” in “Between Term and Definition” selector
  7. Select “New Line” in “Between Definition and Term” selector
  8. Press “Import” button!
